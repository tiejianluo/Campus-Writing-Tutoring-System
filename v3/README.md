# 校园作文辅导系统 v3 📝

## 🎯 版本特色与功能点

### 💡 教育价值
- **激发写作兴趣**：AI智能点评，鼓励性反馈，提升学生写作自信
- **个性化辅导**：根据不同年级、作文类型提供针对性指导
- **减轻教师负担**：自动化评分和点评，节省批改时间
- **促进家校合作**：家长可查看学生写作成长轨迹
- **全面管理**：支持多班级、多学生的统一管理

### ✨ 核心功能

#### 学生端功能
- 📝 **作文创作**：支持多种作文类型（写人、写事、写景、想象作文等）
- 🔍 **智能点评**：AI自动分析作文，根据年级提供分级评分
- 🖼️ **看图作文**：上传图片，生成观察提示和写作指导
- 📚 **范文对比**：与优秀范文对比，学习写作技巧
- 🔄 **分步改写**：提供四步改写指导，循序渐进提升写作水平
- 📈 **成长档案**：记录写作成长轨迹，展示进步趋势
- 🎯 **题目生成**：根据关键词生成作文题目
- 📋 **模板库**：提供各种作文类型的写作框架和句式
- 🔄 **历史版本对比**：对比不同版本的作文，查看改进情况

#### 教师端功能
- 👥 **班级管理**：创建和管理班级，分配学生
- 📋 **作业布置**：批量布置作文题目，设置截止日期
- 📊 **批量查看**：查看所有学生提交的作文和评分
- 🎯 **针对性指导**：根据学生情况提供个性化建议

#### 家长端功能
- 👁️ **查看进度**：查看孩子的写作情况和成长轨迹
- 📈 **成长趋势**：查看孩子的写作能力提升趋势
- 💬 **教师点评**：查看教师对孩子作文的点评

#### 技术特色
- 🔐 **账号系统**：支持教师、学生、家长多角色登录
- 💾 **数据库存储**：使用SQLite数据库存储所有数据
- 🤖 **AI驱动**：基于先进语言模型的智能辅导
- 📱 **界面友好**：简洁直观的操作界面，适合各角色使用
- 📄 **PDF导出**：一键导出作文评语单
- 🔍 **OCR识别**：支持图片文字识别，方便导入手写作文

## 🚀 部署方式

### 📋 环境准备
1. 安装Python 3.8或更高版本
2. 确保网络连接正常（用于接入AI模型）

### 🛠️ 快速部署步骤

#### 第一步：安装依赖
打开命令提示符（CMD）或终端，进入v3文件夹，运行：
```bash
pip install -r requirements.txt
```

#### 第二步：启动应用
在同一文件夹中运行：
```bash
streamlit run campus_essay_system.py
```

#### 第三步：访问应用
启动后，浏览器会自动打开应用页面，或手动访问：`http://localhost:8501`

  - 教师：teacher1 / 123456
