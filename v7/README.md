# 校园作文辅导系统 v7（运营版）

v7 在 v6 基础上修复了人工测试发现的会员额度问题，并新增**后台管理者（运营）功能**：
总体运营看板、账号人工开通/注销、AI 服务状态诊断。

## 相对 v6 的主要变化

1. **"每天 3 次 AI 点评（本地基础点评始终不限次）"语义修正**：
   - v6 中免费额度用完后会直接拦截提交（抛错），学生连基础点评都拿不到；
     v7 改为额度用完后**自动回退到不限次的本地基础点评**，作文照常保存。
   - 点评结果明确标注来源和原因：AI 生成 ✅ / 额度用完 / 系统未配置 AI 密钥 /
     请求限流 / AI 服务出错。v6 中这些情况全部静默回退，人工测试无法区分
     "AI 没配置"和"AI 坏了"。
   - LLM 调用失败现在会写入日志并记录最近错误，管理员端可见。
2. **管理员账号引导**：设置环境变量 `ESSAY_APP_ADMIN_USER` / `ESSAY_APP_ADMIN_PASSWORD`
   后，应用启动时自动创建平台管理员（已存在则跳过），不再依赖演示账号。
3. **管理员端全面升级**，共 6 个标签页：
   - **总体运营**：注册用户数（按角色）、近 7 天新增、作文提交总数/今日、
     AI 点评次数、有效会员数、已核销收入、待核销订单、近 14 天提交趋势图。
   - **账号管理**：账号列表（含状态）、人工开通（创建教师/家长/管理员/学生）、
     **注销**（停用登录、数据保留、可随时重新开通）、重置密码。
     管理员不能注销自己；被注销账号无法登录。
   - **AI 状态**：显示是否已配置密钥、模型与接口地址、最近成功/失败记录，
     并提供"测试 AI 连接"按钮做一次真实调用。
   - 家长绑定、订单核销、激活码（同 v6）。
4. 数据库自动迁移：老库首次启动时自动为 `users` 表补充 `status` 列。

# 使用说明

## 一、安装与启动

```bash
cd v7
pip install -r requirements.txt
streamlit run campus_essay_system.py
```

浏览器打开 `http://localhost:8501`。

### 开启 AI 点评（重要）

系统不配置 AI 密钥也能运行（只有本地基础点评），**要让"每天 3 次 AI 点评"真正
调用大模型，必须配置密钥**。二选一：

环境变量：

```bash
export OPENAI_API_KEY='sk-...'
export OPENAI_BASE_URL='https://api.openai.com/v1'   # 可选，兼容任何 OpenAI 协议服务
export OPENAI_MODEL='gpt-4.1-2025-04-14'             # 可选
```

或 `.streamlit/secrets.toml`（Streamlit Cloud 在 Advanced settings → Secrets 中填写）：

```toml
OPENAI_API_KEY = "sk-..."
```

配置后可用管理员端「AI 状态」页的**测试 AI 连接**按钮验证。

### 创建管理员账号

```bash
export ESSAY_APP_ADMIN_USER='admin'
export ESSAY_APP_ADMIN_PASSWORD='一个至少8位的强密码'
```

启动后即可用该账号登录管理员端。

### 其他可选配置

```bash
export FREE_AI_DAILY_QUOTA=3          # 免费版每天 AI 点评次数（默认 3）
export PREMIUM_PRICE_MONTH=26         # 月度会员价格（元）
export PREMIUM_PRICE_YEAR=288         # 年度会员价格（元）
export PREMIUM_TRIAL_DAYS=7           # 新学生注册赠送会员天数（默认 7，设 0 关闭）
export PAYMENT_QR_MONTH_URL='https://.../qr_month.png'   # 收款二维码
export PAYMENT_QR_YEAR_URL='https://.../qr_year.png'
export ESSAY_APP_DB='essay_campus_system_v7.db'          # 数据库文件路径
```

## 二、各角色使用流程

### 学生

1. 侧边栏「学生注册」：填用户名、姓名、年级、密码；如老师给了**班级邀请码**，
   填上即自动入班。新学生赠 7 天会员体验。
2. 「开始写作文」：选择语文/英语、作文类型和题目（可自定义），参考写作指导，
   写完点击**生成点评并保存**。
   - 会员或有免费额度：AI 智能点评（标注"✅ 本次点评由 AI 智能生成"）。
   - 免费额度用完：自动改为本地基础点评（不限次），作文照常保存。
3. 「继续改写」（会员）：按分步改写指导修改，提交后保存为同一篇作文的新版本。
4. 「历史版本对比」（会员）：任选两个版本左右对比字数与分数变化。
5. 「看图作文」（会员）：上传图片获得 AI 观察提示与启发问题。
6. 「成长档案」：查看练习次数、平均分和成长趋势。
7. 「会员中心」：26 元/月、288 元/年。生成订单号 → 扫码支付并备注订单号 →
   管理员核销后自动开通；也可输入激活码直接开通。

### 老师（账号由管理员开通）

1. 「班级管理」：创建班级，系统生成**邀请码**，发给学生注册时填写。
2. 「布置作文题」：选班级、科目（语文/英语）、类型，填题目、布置说明和截止日期。
3. 「班级提交」：查看本班学生提交的作文、得分和点评。

### 家长（账号由管理员开通，并绑定学生）

登录后选择孩子，查看最近作文、教师点评和成长趋势。

### 管理员

1. 「总体运营」：平台核心指标一览（用户、提交、AI 用量、会员、收入、趋势图）。
2. 「账号管理」：
   - **人工开通**：创建教师/家长/管理员/学生账号（用户名 3-32 位字母数字下划线，
     密码至少 8 位）。
   - **注销**：选中账号后点"注销该账号"，该账号立即无法登录（数据保留）；
     点"重新开通该账号"可恢复。
   - **重置密码**：为忘记密码的用户设置新密码。
3. 「AI 状态」：检查 AI 配置与连通性；学生反馈"没有 AI 点评"时先看这里。
4. 「家长绑定」：把家长账号与学生账号关联。
5. 「订单核销」：核对转账备注中的订单号后点击确认，系统自动开通会员。
6. 「激活码」：生成一次性月度/年度激活码，线下发放。

## 三、免费 / 会员权益

| 功能 | 免费版 | 会员（26 元/月、288 元/年） |
|---|---|---|
| 写作文 + 本地基础点评 | ✅ 不限次 | ✅ 不限次 |
| AI 智能点评 | 每天 3 次 | 不限次 |
| 英语 AI 精批（逐句语法订正） | 计入每日额度 | 不限次 |
| 看图作文 AI 提示 | ❌ | ✅ |
| 继续改写 / 版本对比 | ❌ | ✅ |

说明：只有**真正调用了 AI** 的点评才扣减每日额度；本地基础点评永不扣额度、
永不限次。

## 五、常见问题

**Q：为什么点评结果显示"本地基础点评"而不是 AI？**
点评结果上方会写明原因：未配置 AI 密钥 → 联系管理员配置 `OPENAI_API_KEY`；
今日额度用完 → 明天再来或开通会员；AI 服务出错/限流 → 稍后重试，
管理员可在「AI 状态」页查看具体错误并测试连接。

**Q：升级 v6 数据库会丢数据吗？**
不会。把 `ESSAY_APP_DB` 指向原库即可，v7 启动时自动补充新字段（`users.status`）。

**Q：注销和删除的区别？**
注销只停用登录，作文、订单等数据全部保留，可随时重新开通。系统不提供
物理删除，避免误删学生数据。

## 模块结构

- `app/config.py`：环境变量/secrets 配置（新增管理员引导变量）。
- `app/llm.py`：LLM 调用、回退原因标注、错误记录、连接测试。
- `app/services.py`：业务服务（新增额度回退语义、账号管理、运营统计）。
- `app/storage.py`：SQLite 存储（新增 `users.status` 迁移与 `dashboard_stats`）。
- `app/ui.py`：Streamlit 界面（管理员端 6 标签页）。
- `testcode/`：单元 + 系统 + 验收测试（70 个）。
